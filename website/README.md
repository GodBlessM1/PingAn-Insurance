# 平安寿险收益分析网站

## 网站文件位置
- 主页面: `index.html`
- 图表文件: `charts/` 目录

## 如何打开网站

### 方法1：直接双击打开（推荐）
直接双击 `index.html` 文件，会在默认浏览器中打开

### 方法2：使用启动脚本
双击 `启动网站.bat` 文件，会启动本地服务器
然后在浏览器中访问: http://localhost:8080

### 方法3：手动启动服务器
在命令行中执行:
```bash
cd website
python -m http.server 8080
```
然后在浏览器访问: http://localhost:8080

## 网站内容

### 5个可交互页面
1. **首页概览** - 核心数据统计和关键发现
2. **数据分析** - 图表展示和趋势分析
3. **产品分析** - 四大产品类型详细对比
4. **年度趋势** - 时间序列分析和预测
5. **分析报告** - 完整的分析报告

### 功能特性
- 响应式设计，适配各种屏幕
- 平滑动画效果
- 交互式导航
- 可折叠的报告章节
- 数据可视化图表

## 数据来源
- 保单数据: 1,000份保单（2020-2024年）
- 产品类型: 传统寿险、分红险、万能险、投连险
- 分析指标: IRR、年化回报率、总回报率

## 详细分析报告
查看 `reports/详细分析报告.md` 获取完整的分析报告

## 技术支持
如有问题，请检查:
1. Python是否已安装
2. 浏览器是否支持HTML5
3. 图表文件是否存在于 charts/ 目录
